Project Title: TurtleBot3 Navigation and Mapping

Author: Meghna Pradeep
Institution: Muthoot Institute of Technology and Science

Description:
This ROS-based project demonstrates SLAM and autonomous navigation using TurtleBot3 in Gazebo.
It includes world creation, mapping, localization, and navigation phases.

Contents:
- launch/ : Contains all launch files (slam, navigation, final_project)
- maps/ : Contains saved final map (.pgm, .yaml)
- worlds/ : Includes my_custom_world.world and turtlebot3_world.world
- phase4_demo.bag : Recorded navigation data
- package.xml, CMakeLists.txt : ROS package setup files
